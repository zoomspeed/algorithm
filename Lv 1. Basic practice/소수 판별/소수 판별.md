## 소수 판별
***
### 문제
***
자연수n이 주어질 때, n 이 소수인지 판별하는 프로그램을 작성하여라. 여기서 소수란, 약수가 1과 자기자신밖에 존재하지 않는 수를 말한다.

### 입력
***
첫째 줄에 자연수 n이 주어진다. (1≤n≤20,000)

### 출력
***
첫째 줄에 n이 소수이면 YES, 소수가 아니면 NO를 출력한다.

 

### 예제입력
```
2
```

### 예제출력
```
YES
```

### 예제입력
```
13
```

### 예제출력
```
YES
```

### 예제입력
```
15847
```

### 예제출력
```
NO
```
